# Progress Tracker – Psychology-Based Learning Platform

Spring Boot приложение для отслеживания прогресса обучения с психологическими метриками.

## Технологии
- Java 17
- Spring Boot
- PostgreSQL
- Maven
- Lombok

## Запуск
1. Создайте базу данных PostgreSQL.
2. Настройте подключение в `application.yml`.
3. Выполните `mvn spring-boot:run`.
